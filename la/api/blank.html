LightApp 基础框架 API

方法列表

方法名称 方法说明
[function] init(pagename, options) 页面初始化，参数pagename 为页面名称，options 为页面的参数配置
[function] gotoPage(url)

页面跳转，采用iframe预加载模式跳转页面

自动对页面的所有a标签绑定该方法，可以通过设置data-async="1"来阻止页面跳转
[function] ajax(options)

页面跳转，采用iframe预加载模式跳转页面

自动对页面的所有a标签绑定该方法，可以通过设置data-async="1"来阻止页面跳转
[function] loading() 显示loading页面

DOM 节点参数配置

参数名称 参数说明
data-async 配合LightPage的跳转策略;设置为1，将跳转策略改成不跳转，异步获取数据
data-page 设置下一页的页码

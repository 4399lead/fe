幻灯片组件 API

参数列表

参数名称 参数说明
[object]options 参数配置

options 参数

参数名称 参数说明
[selector] target 滚动items的容器 例如 ul
[selector] nav_target 导航容器
[selector] items 滚动itms 例如 li
[bool] hasTap 是否包含链接
[int] margin 滚动items的间距，css也需要设置
[int] width 图片宽度
[float] ratio 图片宽高比例
[int|string] scrollNum 一次滚动数量，如设置 auto 表示自适应个数
[int] delay 自动播放间隔时间 单位毫秒
[int] speed 每次滚动的时间 单位毫秒
[bool] autoplay 是否自动播放

项目构建

必备软件：

本套构建包基于node.js环境。

nodeJS下载、Git Bash下载

安装步骤：

1：为了保证项目构建集成包统一管理，托管在github上，获取基础构建包时，请在git Bash中操作.

命令行:git clone https://github.com/f2er/grunt-init-ks.git ~/.grunt-init/ks

2：初始化项目模板，先确认本机中是否安装grunt-init，如没安装，执行 npm install -g grunt-init

3：切换到本地项目目录,如project，执行 grunt-init ks，其他按提示操作即可或直接Enter跳过.安装成功后，执行 npm install 直接回车，进入安装grunt 插件。整个前端基础构建包完成。

集成包功能：

包含CSS合并压缩、JS混淆压缩、模板页引入、图片优化压缩等。

集成包功能插件：

grunt-contrib-concat

grunt-contrib-cssmin

grunt-contrib-imagemin

grunt-contrib-uglify

grunt-includes

grunt-contrib-watch

静态资源管理：采用Bower包管理工具

如本机中没有安装bower，可以在项目目录，在git Bash中操作 npm install bower -g

Bower 资源列表：前端组件列表

通过bower除了可以下载官方上的插件外，还可以下载本地组件、团队线上组件。只要通过简单的命令就可以实现资源统一管理 bower install path/to/**.js

更新：每需要一个插件就bower install就很繁琐，现在只要对项目中的bower.json文件中dependencies对象做一定的修改即可，最后执行一次bower install即可安装项目所需要的插件。

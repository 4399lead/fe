滚动条组件 ue.scrollbar

参数列表

参数列表
参数名称 备注
width / height 设置width 表示横向滚动和内容区域的可见大小；设置height 表示垂直滚动和内容区域的可见大小
scrollbar_width / scrollbar_height 滚动条的高度或者宽度 默认值是内容的宽高
available_scroll_size 滚动条的可以滚动的范围，默认为滚动条的高度
scroll_per 每次滚动滑轮，滚动条移动的距离；默认是 20
target 含有滚动条的容器
box 包含内容的容器
scrollbarbg 滚动条背景
scrollbar 滚动条容器
btn 滚动条按钮
onscroll 滚动条滚动式触发回调函数

方法列表

方法列表
方法名称 备注
reset
重新设置滚动条的大小
animate 是否支持动画效果 默认是 false
speed 滚动的速度 默认是 100
scrollTo
滚动滚动条到指定的位置
pos 滚动条要滚动到的位置 支持 数字 和百分百
animate 是否支持动画效果 默认是 true
speed 滚动的速度 默认是 200
silent 滚动是否触发onscroll事件 默认是false
isstop 是否先停止当前的运动再开始滚动 默认是false
callback 滚动完的回调函数 默认是 noop
boxScrollTo
滚动内容区域到指定的位置
pos 滚动条要滚动到的位置 支持 数字 和百分百
animate 是否支持动画效果 默认是 true
speed 滚动的速度 默认是 200
silent 滚动是否触发onscroll事件 默认是false
isstop 是否先停止当前的运动再开始滚动 默认是false
callback 滚动完的回调函数 默认是 noop

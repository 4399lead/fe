静态分页组件 ue.pager

参数列表

参数列表
参数名称 备注
formTarget 放置表单分页跳转jquery对象
pagerTarget 放置表单分页页码对象
first 首页的模版，默认为<a href="#" class="pager_first pager_item">首页</a>
firstDisabled 首页不可点击的模版，默认为<span class="pager_first pager_item">首页</span>
last 末页的模版，默认为<a href="#" class="pager_last pager_item">末页</a>
lastDisabled 首页不可点击的模版，默认为<span class="pager_last pager_item">末页</span>
prev 上一页的模版，默认为<a href="#" class="pager_prev pager_item">上一页</a>
prevDisabled 上一页不可点击的模版，默认为<span class="pager_prev pager_item">上一页</span>
next 下一页的模版，默认为<a href="#" class="pager_next pager_item">下一页</a>
nextDisabled 下一页不可点击的模版，默认为<span class="pager_next pager_item">下一页</span>
current 当前页的模版，默认为<span class="pager_page pager_item pager_current">@{page}</span>
page 数字页码的模版，默认为<a href="#" class="pager_page pager_item">@{page}</a>
tip 分页信息的模版，默认为<span class="pager_item pager_tip">@{nowPage}/@{pageCount}</span>
goto 表单分页跳转的模版，默认为<form><input class="pager_input"/><input type="submit" value="go" class="pager_goto"/></form>
gotobtn 调整按钮的选择符，默认为.pager_goto
input 页码输入框的选择符，默认为".pager_input"
now 当前显示的页码，默认是1
maxPage 显示最大数字页码的个数，默认为5
per 每页的记录条数，默认为5
count 所有记录的总数
onchange 点击翻页时的回调函数，传递的第一个参数为将要跳转的页面的页码
新增
showFirstAndLastNum 是否显示数字的首页末页 即 1代表首页 总页数代表末页 默认是 false
firstNum 首页数字模式的模版，默认为<a href="#" class="pager_firstnum pager_item">1</a>
firstNumDisabled 首页数字模式不可点击的模版，默认为<span class="pager_firstnum pager_item">1</span>'
lastNum 末页数字模式的模版，默认为<a href="#" class="pager_lastnum pager_item">@{pageCount}</a>
lastNumDisabled 首页数字模式不可点击的模版，默认为<span class="pager_lastnum pager_item">@{pageCount}</span>
showMore 是否显示更多翻页 默认是 false
prevMore 向前更多页的模版，默认为 : <a href="#" class="pager_prevmore pager_item">...</a>
prevMoreDisabled 向前更多页不可点击的模版，默认为<span class="pager_firstnum pager_item">1</span>'
nextMore 向后更多页的模版，默认为<a href="#" class="pager_nextmore pager_item">...</a>
nextMoreDisabled 向后更多页不可点击的模版，默认为<span class="pager_nextmore pager_item">...</span>

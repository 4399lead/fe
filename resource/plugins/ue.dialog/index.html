基础对话框 ue.dialog

基础对话框的参数列表
参数名称 备注
id 对话框的id。参数值为ID名称
force 如果已经存在相同id的对话框 是否强制关闭 ，参数值格式 false | true
content 对话框的内容
zIndex 对话框的zIndex
lock 是否显示遮罩层 参数值 ： false | true
width 设置对话框的宽度，默认值为auto，即自适应模版的宽度。参数值格式为 ： 数字 | auto | 百分数("100%")
height 设置对话框的高度，默认值为auto，即自适应模版的高度。参数值格式为 ： 数字 | auto | 百分数("100%")
minWidth 设置对话框的最小宽度，默认值为auto，即不限制最小宽度。参数值格式为 ： 数字 | auto
maxWidth 设置对话框的最大宽度，默认值为auto，即不限制最大宽度。参数值格式为 ： 数字 | auto
position 设置对话框的定位的模式，默认值为absolute。参数值为 absolute | fixed
left 设置对话框的位置。默认值为auto,即居中。参数值为 数字 | "auto"
right 设置对话框的位置。默认值为auto,即居中。参数值为 数字 | "auto"
top 设置对话框的位置。默认值为auto,即居中。参数值为 数字 | "auto"
bottom 设置对话框的位置。默认值为auto,即居中。参数值为 数字 | "auto"
drag 是否可以拖动 参数值: true | false
dragHock 设置可以拖动的元素 参数值：[string]jquery 选择符
closeBtn 关闭对话框按钮的class，默认值 ".ui-dialog-close_btn"
init 设置对话框初始化的方法
beforeClose 关闭弹窗之前的回调函数
afterClose 关闭弹窗之后的回调函数
triggerTarget 触发弹窗的对象，用于关闭弹窗后继续把焦点转移给它
基础对话框的静态成员列表
参数名称 备注
list 当前所有存在的对话框的列表，可通过id去获取，如 uiDialog.list["id"]
maskClosable 设置蒙层是否可以点击关闭
基础对话框的公共成员列表
参数名称 备注
obj 当前对话框的jquery对象，如 this.obj或者 uiDialog.list["id"].obj
基础对话框的公共方法列表
参数名称 备注
reset

调整对话框内容后调用此方法重置高度和位置
close([beforeClose, afterClose]) 关闭对话框并删除，同时可以传新的beforeClose 和 afterClose 回调
show 显示对话框
hide 隐藏对话框
timer

设置定时器

参数列表
delay,设置定时器的事件 1000 表示1秒
callback 设置定时器触发的函数

具有默认主题的对话框 ue.dialog.custom

基础对话框的参数列表
参数名称 备注
show_footer 是否显示按钮区 默认 false 参数值 true | false
show_cancel_btn 是否显示取消按钮 默认值 false 参数值格式 false | true
show_confirm_btn 是否显示确认按钮 默认值 false 参数值格式 false | true
title 对话框的标题
content 对话框主体内容
footer 自定义按钮区内容
theme 对话框的主题 默认值 'ui-dialog-a'
show_header 是否显示头部 默认 true 参数值 true | false
confirm 点击确定按钮的回调函数
cancel 点击取消按钮回调函数
delay 设置对话框定时关闭的时间 默认 是0 即 没有启动定时关闭功能
icon 设置uiDialog.Alert、uiDialog.Confirm、uiDialog.Tip 的 左侧图标对应的className，目前内置有三种icon ui-dialog-failure_icon、ui-dialog-add_icon、ui-dialog-info_icon,可以通过自己定义icon，并将iconclassName 当做参数传递
其他的参数 其他的参数参照 uiDialog

跟随对话框 ue.dialog.follow

基础对话框的参数列表
参数名称 备注
options 传递一个 uiDialog的参数对象，用于创建一个弹层
isfollowed 被跟随的jquery对象
pos 跟随的位置,水平方向 有 left center right 竖直方向有 top middle bottom 参数为 两个方向的组合 比如"left_right" ，该组件也会对对话框添加一个 对应的classname 如（left_right）以便于可以通过css 去控制 自定义箭头的指向
trun 跟随位置的微调 比如 {x : 3, y :-5} 表示 在pos 控制位置的基础上 再往右移动三个像素，往上移动5个像素

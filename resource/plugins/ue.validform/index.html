表单验证组件 ue.validform

参数列表

参数列表
参数名称 备注
form 表单对象的选择符，如"#j-form1"
onsubmit 表单提交时的回调函数，传递的第一个参数为当前表单的验证结果；如果该回调函数 return true 则会直接提交，如果return false 则不会提交
failureTipClass 验证结果为不通过时，给提示对象添加的class
successTipClass 验证结果通过时，给提示对象添加的class
elements 存放需要验证的表单元素的配置数组

输入框（文本框、密码框、多行输入框）的配置数组说明

输入框（文本框、密码框、多行输入框）的配置数组说明
参数名称 备注
selector 输入框对象的选择符，如"#j-username"
tip 输入框对应的提示对象的选择符，如"#j-username_tip"
trim 是否先对输入框的输入值先进行去除前后空白再进行验证，默认值是true
required 输入框是否为必填，默认为 false
requiredText 输入框验证为空时的提示信息，如"<i>×</i>用户名不能为空"；如果没有配置则使用failureText的值
minLength 输入框最少输入长度限定
maxLength 输入框最多输入长度限定
lengthType 输入长度限定的模式，有ch 多少个字，en 多少个字符 两种模式，默认为 en
lengthText 输入长度超出限定时的提示文字；如果没有配置则使用failureText的值
regexp 输入框用正则表达式进行验证，填写正则表达式对象
regexpText 正则表达式验证不通过的提示文字；如果没有配置则使用failureText的值
relation 关联验证的对象的选择符
relationType 关联验证的类型 有 ==、>=、<=、>、<、!=
relationText 关联验证不通过时的提示文字；如果没有配置则使用failureText的值
email 启用email验证，默认为false
emailText email验证不通过的提示信息；如果没有配置则使用failureText的值
failureText 所有验证不通过时的默认提示信息
successText 验证通过时的提示信息
onvalid 验证结束后的回调函数，可用于做其他的业务级别的验证；传递的第一个参数为当前输入框的验证结果
after 前置验证对象，前置验证必须为真才会验证改表单
before 验证条件，验证条件必须为真才会验证改表单

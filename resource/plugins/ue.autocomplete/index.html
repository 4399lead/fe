自动完成框组件 ue.autocomplete

参数列表

参数列表
参数名称 备注
input 输入框的jquery 选择符 如 ：".ue-autocomplete-input input"
target 下拉框的jquery 选择符 如 ：".ue-autocomplete-list"
list 下拉框中放置选项的对象的选择符 如 ：".ue-autocomplete-list ul"
items 下拉框中选项的选择符 如 ：".ue-autocomplete-list li"
currentClass 下拉框选项被选中的classname 默认为 "cur"
auto 当auto 为 true时，通过移动上下键选择下拉框选项的值时或者点击下拉框选项时，都会将下拉框选项对象的属性 data-value 的值赋给输入框；为 false 则不改变输入框的值；第一个参数值为输入框的值
handle 控制下拉框显示的回调函数，用于执行匹配和搜索。
onSubmit 点击下拉框选项或者按回车键时的回调函数；第一个参数值为当前点击或者触发回车键的选项

方法列表

方法列表
方法名称 备注
hide 隐藏下拉框
show
显示下拉框
html 下拉框要显示的列表项的html
